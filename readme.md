#pymaScreen - Screen pixel color reading on Mac OS X using built-in python (no helper apps necessary)

pymaScreen is a single-file python library that can be used with the built-in python on OS X to get information about pixels on the screen. This is performed without need for additional helper applications through the use of pyObjC.
